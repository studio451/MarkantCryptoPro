# MarkantCryptoPro #

Использование КриптоПРО.NET (https://www.cryptopro.ru/products/net) для формирования марканта (структуры для передачи зашифрованного сессионного ключа и вспомогательной информации)

Использование тестового приложения:
1. Создаем сессионный ключ
2. Шифруем текст на сессионном ключе
3. Выбираем алгоритм шифрования сессионного ключа в марканте
4. Создаем маркант на сессионном ключе
5. Сбрасываем сессионный ключ
6. На "принимающей" стороне извлекаем сессионный ключ из марканта
7. Дешифруем текст с помощью извлеченного сессионного ключа

В проекте использованы компоненты:
https://github.com/Asn1Net/Asn1Net.Reader
https://github.com/Asn1Net/Asn1Net.Writer
